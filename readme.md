# Online Book Shop MERN APP

This is the online book shop app with a recommender system implemented via Flask REST API

## Usage

Rename the .envexample to .env and add your MONGO_URI

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run dev
```
